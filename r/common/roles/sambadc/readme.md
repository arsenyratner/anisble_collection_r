# Описание роли

Роль для развёртывания контроллеров домена на базе samba. Если используется DNS бэкенд BIND9_DLZ, то необходимо настроить bind

* проверяет, развёрнут ли домен уже?
* устанавливает пакеты
* развёртывает первый КД если sambadc_deploy_firstdc is true
* добавляет ещё один КД в уже существующий домен если sambadc_deploy_adddc is tru
* копирует smb.conf, krb5.conf
* перезагружает компьютер
